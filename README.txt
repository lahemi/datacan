
Save and display simple text data, at the moment
meant mainly for links, through a web browser.

Uses mattn's go-sqlite3 package.

Written and tested using Go 1.3.

Uses $HOME/.local/share/datacan for html,
CSS and storing the database file.


Command line options:
    -db     which you can use to specify the SQLite3 db file to use.
    -port   which you can use to change the default 19999 port.
